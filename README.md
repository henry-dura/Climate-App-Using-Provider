# 📖 CLIMATE APP  <a name="about-project"></a>

**Climate App** is designed using the Provider state management. It displays loading indicator while waiting for weather information. Gets weather information from Open Weather Map using location or city name and displaying weather information in a beautiful UI.

## 🛠 Built With <a name="built-with"></a>

This project was built using Flutter framework,Provider state management, and dart programming language. 



<!-- Features -->

### Key Features <a name="key-features"></a>

- **Show Loading indicator when waiting for weather data from Open Weather Map**
- **Get weather data using location Longitude and Latitude**
- **Search for weather data for any city of choice**
- **Get advise on the best outfit for the weather**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:
- A personal computer connected to the internet
- Have a local version control like git installed on your computer and a Github account.
- Install flutter on your machine: [Flutter Installation Guide](https://docs.flutter.dev/get-started/install)
- Setup your code editor (Visual Studio Code, Android Studio) :  [Editor Setup](https://docs.flutter.dev/get-started/editor)
- Run `flutter doctor` to confirm everything is setup properly.


### Setup

Clone this repository to your desired folder:

```sh
git clone https://github.com/henry-dura/Climate-App-Using-Provider
```
```sh
cd Climate-App-Using-Provider
```


### Usage

To run the project in your emulator, execute the following command:

```sh
flutter pub get
```

```sh
flutter run
```
